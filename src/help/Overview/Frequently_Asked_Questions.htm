Frequently Asked Questions

What is Language Forge?

Language Forge is online dictionary software. That is, it is a tool you can use in any Internet browser to work with lexical data.

  • See Also: https://languageforge.org/learn_language_forge

What is Language Depot?

Language Depot is an online storage server that keeps the repositories of language data that you work with by way of Language Forge, FieldWorks Language Explorer (FLEx) and other language software.

  • See Also: https://public.languagedepot.org/

How does Language Forge work with Language Depot?

Language Forge is an online tool that allows you to work with the language data you have in Language Depot, which is the repository or "depot" for language data. The various controls in Language Forge allow the owner of the project to determine who can see the lexical data, and then for each person, roles are assigned so there are controls regarding which of the data they can see or change.

How does FieldWorks Language Explorer work with Language Depot and Language Forge?

For a language project, consider the following:

  • FieldWorks Language Explorer (FLEx) uses Send/Receive features to synchronize the lexical data in FLEx with the repository in Language Depot. Send/Receive is manually initiated by each person who collaborates in a language project by using FLEx.

  • For Language Forge users, a project manager manually initiates each Send/Receive operation to synchronize the lexical data with the repository in Language Depot. In contrast to FLEx, within Language Forge all collaborators see each other's updated in nearly real time. This allows them to collaborate together in a dictionary project without doing any Send/Receive.

The project manager initiates the Send/Receive operation to synchronize Language Depot and Language Forge to ensure that Language Forge users get the most recent changes done by FLEx users, and FLEx users get the most recent changes done by Language Forge users. Because publication is done by way of FLEx, it is necessary that FLEx has the most current data set.
